# Security Policy

## Credential Rotation

Production database credentials must be rotated every 90 days. Emergency credential rotation is required immediately after suspected compromise or employee offboarding from a privileged role.

## Data Access

Employees may only access customer data for approved support, engineering, compliance, or security purposes. Access to regulated customer data requires manager approval and audit logging.

## Device Requirements

Company laptops must use full-disk encryption, automatic screen lock, endpoint protection, and operating system updates within 14 days of release.
